# HomeLab — Rancangan Infrastruktur Fintech / Digital Banking

Laboratorium infrastruktur on-premise untuk mensimulasikan stack produksi **fintech** atau **digital banking**:
transaksi, ledger, event streaming, cache sesi, pencarian, observability, dan delivery berbasis GitOps — di cluster Proxmox 5 node dengan resource terbatas (~80 GB RAM total).
